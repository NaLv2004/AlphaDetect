\section{AlphaDetect：基于中间表示的MIMO检测算法自动发现}
% ============================================================

本周完成了 AlphaDetect 框架核心模块的实现，包括：(1) 面向MIMO检测算法的编译器中间表示（IR）设计与编译前端；(2) 以IR为操作对象的双层进化引擎，结合图神经网络（GNN）强化学习实现算法结构的智能嫁接。

\subsection{算法中间表示（FunctionIR）}

我们设计了一套静态单赋值（SSA）形式的算法中间表示 \texttt{FunctionIR}，将任意Python MIMO检测函数编译为可操作的图结构。如图~\ref{fig:ir}所示，IR由三类元素构成：
\begin{itemize}
  \item \textbf{Value}：数据节点，每个Value唯一由一个Op定义（SSA性质），可被多个Op消费，携带类型标注（标量/向量/矩阵）；
  \item \textbf{Op}：操作节点，包含操作码（\texttt{binary}、\texttt{call}、\texttt{const}等），通过输入/输出Value列表显式描述数据流；
  \item \textbf{Block}：控制流基本块，多个Block通过前驱/后继关系构成控制流图（CFG）。
\end{itemize}

此外，IR中引入了 \textbf{AlgSlot} 机制——一类特殊的 \texttt{call} 操作，表示算法中可抽象替换的功能模块（黑盒）。SlotPopulation 维护每个 Slot 的一组实现变体，供微观层进化独立优化。

\subsection{双层进化引擎与GNN引导的骨架嫁接}

基于FunctionIR，我们实现了双层进化引擎（\texttt{AlgorithmEvolutionEngine}）：
\begin{itemize}
  \item \textbf{宏观层（Macro）}：操作算法骨架，通过PatternMatcher识别宿主（Host）IR和供体（Donor）IR中的可替换区域（Region），执行骨架嫁接（Grafting），产生跨算法族的混合子代；
  \item \textbf{微观层（Micro）}：在每个AlgSlot的SlotPopulation内部，通过IR变异（\texttt{mutate\_ir}）和交叉（\texttt{crossover\_ir}）进行局部优化；
  \item \textbf{评估层}：将任意AlgorithmGenome物化（Materialize）为可执行Python函数，通过MIMO Monte Carlo仿真计算误符号率（SER）作为适应度。
\end{itemize}

宏观层的核心创新是 \textbf{GNN模式匹配器}（\texttt{GNNPatternMatcher}）。如图~\ref{fig:graft}所示，该模块将Host和Donor的FunctionIR分别转化为图神经网络输入（节点=Op，边=数据流+控制流），通过图注意力网络（GAT）学习图嵌入，评分所有Host-Donor配对，并利用REINFORCE算法以嫁接后SER提升作为奖励信号在线更新网络参数。嫁接操作在IR级执行：克隆宿主IR，在宿主中插入对供体的\texttt{call}操作，重绑定输出，最终通过拓扑排序保证SSA合法性。
